About the Project

This tool is designed for Nigerian farmers and agricultural advisors. It allows users to upload an image of a crop leaf, identifies the disease using an AI model trained on common crop diseases (like maize rust, rice blast, and potato blight), and then suggests treatment options using real agricultural data from FAO AGRIS and LLaMA 3 AI model via OpenRouter.
